# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it
responsibly by emailing **rob.plenee@gmail.com**.

Do not open a public GitHub issue for security vulnerabilities.

We will acknowledge receipt within 2 business days and provide a resolution
timeline based on severity per our patch SLA:
- Critical: 24 hours
- High: 7 days
- Medium: 30 days
